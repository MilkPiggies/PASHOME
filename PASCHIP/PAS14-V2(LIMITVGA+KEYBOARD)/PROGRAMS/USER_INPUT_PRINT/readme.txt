This program (USER_INPUT_PRINT) does the following:

-Sets the display to FG=WHITE, BG=BLUE.
Draws a bar on the first line, filling the line with "-".
Draws "Input: " on the second line.
Sets KB pointer to second line, third column.
Waits for the user to type a string.
once user inputs a string, draws what the user input was on the second line after "Input: ".
